# Автономный брутфорс несжатых глифов — как запустить

Ищет НЕсжатые (raw A1/A2/A4/A8) глифы/иконки/картинки в дампах, скорит по «глифо-подобности»,
сохраняет РАНЖИРОВАННУЮ галерею PNG (лучшие первыми) + CSV с адресами. Автономно, без надзора.

## Запуск (одна команда, оба дампа)

```bash
cd C:\Git\TKMI-SPDT\other\geely-coolray-dashboard-translation
python tools/brute_raw_glyphs.py
```

Это прогонит U17 и U18 подряд. Оценка времени: **~3 часа суммарно** (U17 ~1.8ч, U18 ~1.2ч).
Можно оставить на ночь. Прогресс пишется в `review/brute/progress_*.txt` — можно подсматривать.

## Что получишь

```
review/brute/Coolray_25_Dash_U17/
    catalog.csv              ← все находки: rank, score, offset, format, width, bitorder, ...
    gallery/
        0001_s0.877_028FEFB0_A8_w96_msb.png   ← лучшие с rank 0001 (score в имени)
        0002_...
review/brute/Coolray_25_Dash_U18/  (то же для второго чипа)
```

Картинки отсортированы: **начинай смотреть с 0001** — там самые перспективные. Имя = ранг +
score + hex-адрес в дампе + формат (A8/A4/A2/A1) + ширина + бит-порядок.

## На что смотреть глазами
- **Ряд одинаковых по стилю форм** = атлас шрифта (буквы/цифры/иероглифы) — главный приз.
- **Одиночная узнаваемая форма** = иконка/пиктограмма.
- **Полосы/градиент/каша** = ложное срабатывание (RLAD просочился) — игнорируй.
- Нашёл что-то стоящее? Запомни hex-адрес из имени — по нему прицельно до-вытащим ресурс.

## Опции (если надо)
```bash
# крупная графика (не глифы) — увеличить окно высоты:
python tools/brute_raw_glyphs.py --height 64

# строже/мягче порог (меньше/больше картинок):
python tools/brute_raw_glyphs.py --min-score 0.45

# один дамп:
python tools/brute_raw_glyphs.py --dumps Coolray_25_Dash_U17.bin
```

## Прерывание/резюм
Ctrl+C безопасен — сохранённое цело. Повторный запуск ПРОПУСКАЕТ дампы, где уже есть
`catalog.csv` (удали его, чтобы пересчитать заново).

## Важное ограничение (честно)
Работает ТОЛЬКО по НЕсжатым зонам (distinct bytes < 64). RLAD-сжатый контент (цвет, вероятно
текст-шрифт) этот инструмент НЕ берёт by design — там нужен старт+ширина из таблицы MCU
(JTAG XIP) или diff двух языков. См. docs/INVESTIGATION_HISTORY.md §3a.
